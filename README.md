# HabitPulse

A simple daily habit tracker for freelancers.

## About

Freelancers lack structure — no boss, no schedule, habits slip without accountability. HabitPulse helps freelancers build better routines by tracking daily habits and building streaks.

## Tech Stack

- **Framework:** Next.js 15 (App Router) + TypeScript
- **Database:** Supabase (PostgreSQL + Auth + RLS)
- **Styling:** Tailwind CSS + shadcn/ui
- **Deployment:** Vercel
- **Analytics:** PostHog

## Getting Started

### Prerequisites

- Node.js 20+
- npm
- Supabase account

### Setup

1. Clone this repository
2. Copy `.env.example` to `.env.local` and fill in values
3. Run `npm install`
4. Run `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)

### Environment Variables

See `.env.example` for all required variables.

## Development

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run lint` — Run ESLint
- `npm run test` — Run tests
